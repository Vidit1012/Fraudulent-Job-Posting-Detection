# Fraudulent Job Posting Detection

> Academic Project · Introduction to Pattern Recognition · University at Buffalo  
> **Team:** Anirudh Mhaske · Dhiraj Patil · Vidit Rajesh Prabhu · Hemasree Pujari

---

## Overview

A supervised learning pipeline to detect fraudulent job postings from a real-world dataset of ~18,000 listings. Addresses severe class imbalance (only 5% fraudulent) through resampling and feature engineering.

---

## What It Does

- Cleaned and preprocessed real-world job listing data — handled 70k+ missing values, outliers, label encoding, and normalization
- Trained and evaluated 5 models: Decision Tree, Random Forest, Gradient Boosting, Logistic Regression, and SVM
- Best accuracy of **98.69%** (Decision Tree) with Random Forest achieving zero false positives
- Generated correlation heatmaps and visualizations to identify key fraud indicators
